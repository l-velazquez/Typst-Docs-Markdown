
# Boolean

A type with two states.

The boolean type has two values: <span class="typ-key">`true`</span> and
<span class="typ-key">`false`</span>. It denotes whether something is
active or enabled.

## Example

<div class="previewed-code">

    #false \
    #true \
    #(1 < 2)
