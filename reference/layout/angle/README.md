
# Angle

An angle describing a rotation.

Typst supports the following angular units:

- Degrees: <span class="typ-num">`180deg`</span>
- Radians: <span class="typ-num">`3.14rad`</span>

## Example

<div class="previewed-code">

    #rotate(10deg)[Hello there!]
