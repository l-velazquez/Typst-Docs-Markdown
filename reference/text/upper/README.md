
# upper

```
upper(
    str content
) -> str content
```
Converts a string or content to uppercase.

## Example

<div class="previewed-code">

    #upper("abc") \
    #upper[*my text*] \
    #upper[ALREADY HIGH]

### Parameters


### text: str, content | _required_ _positional_

The text to convert to uppercase.
