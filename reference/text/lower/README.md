
# lower

```
lower(
    str content
) -> str content
```
Converts a string or content to lowercase.

## Example

<div class="previewed-code">

    #lower("ABC") \
    #lower[*My Text*] \
    #lower[already low]

### Parameters


### text: str, content | _required_ _positional_
